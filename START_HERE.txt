🎯 WHAT IS THIS?
─────────────────
A complete Node.js automation framework that lets you automate browser
interactions by writing simple JSON files instead of code.

No programming required - just define steps in JSON and run!


📦 INSTALLATION (First Time Setup)
────────────────────────────────────
1. Open terminal in this directory
2. Run: npm install
3. Run: npm run install-browsers
4. Done! (Takes 2-5 minutes)


🚀 QUICK START
────────────────
Run the demo (no configuration needed):

    npm run demo

This will:
- Open a browser
- Search Google
- Take screenshots
- Show you a complete automation in action!

──────────────────────────
1. Copy an example rule file

4. Check screenshots/ and logs/ folders for results


🛠️ AVAILABLE COMMANDS

💾 HOW IT WORKS
────────────────
1. You create a JSON file with automation steps
2. Each step defines an action (click, type, etc.)
3. Run the engine with your JSON file
4. Engine executes steps in order
5. Results are logged and screenshots captured
6. Done!

Example step:
{
  "stepId": "1",
  "action": "navigate",
  "enabled": true,
  "url": "https://example.com"
}

🆘 TROUBLESHOOTING
───────────────────
Browser doesn't open?
→ Set HEADLESS=false in .env file

"Element not found" error?
→ Add waitForSelector before interacting
→ Verify selector in browser DevTools

Navigation timeout?
→ Increase NAVIGATION_TIMEOUT in .env

Browsers not installed?
→ Run: npm run install-browsers
