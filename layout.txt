Browser Automation Platform** - Developed a full-stack web automation platform using Angular 19, Node.js, Express.js, and Playwright that enables users to create and execute browser automation workflows through an intuitive visual interface. The platform features real-time execution monitoring via WebSocket, a no-code rule builder, and supports concurrent automation execution across multiple browsers. Implemented RESTful APIs, WebSocket communication, and a modular architecture supporting multi-environment deployments.
